<h1 align="center">Music Downloader</h1>

<p align="center">Herunterladen von Musik über einen YouTube oder Spotify Link als MP3-Datei. Die 
Lyrics der Songs werden in den Metadaten der MP3-Dateien gespeichert.</p>
